# Hiro Quest — Roulette & Quests (Test Task)

Интерактивная промо-страница с механикой "Колесо фортуны", выполненная в рамках тестового задания.

## 🚀 Ссылка на демо

[Посмотреть проект на Vercel](https://hiro-quest-roulette.vercel.app/)

## 🛠 Стек технологий

- **React 18** + **TypeScript** (строгая типизация всех компонентов и данных).
- **Vite** — быстрая сборка и разработка.
- **Framer Motion** — сложная анимация рулетки (плавное замедление через cubic-bezier), модальных окон и прогресс-бара.
- **Canvas Confetti** — визуальные эффекты при выигрыше.
- **CSS Variables** — централизованное управление цветами и шрифтами.

## ✨ Ключевые особенности

- **Реалистичная рулетка:** Расчёт остановки точно по центру выигрышного элемента, статичная рамка-фокус и индикатор.
- **Smart-анимация:** Использование инерции (ease-out) для имитации физического барабана.
- **Адаптивность:** Полная поддержка мобильных устройств, планшетов и десктопов.
- **Таймер обратного отсчёта:** Реализован живой отсчёт времени после завершения игры.
- **Прогресс-бар:** Анимированная полоса достижений за 7 дней.

## 🤖 Использование ИИ

В соответствии с требованиями ТЗ: при выполнении задания использовался **ChatGPT (GPT-4o)** в качестве ментора для:

- Помощи в настройке окружения и дебага путей при деплое.
- Расчёта математической формулы центрирования рулетки относительно окна.
- Консультаций по строгой типизации пропсов в TypeScript.
